Trang Chủ Trang Quản Lý Thử Thách Lập Trình (Debug)

Yêu cầu: Tìm và sửa lỗi hàm deleteRow(index)

Mô tả bài tập: Sinh viên cần đưa đoạn mã gốc gặp lỗi logic vào trang web, tiến hành phân tích các điểm sai và đưa ra khối mã JavaScript hoàn chỉnh đã khắc phục thành công bên dưới.
1. Đoạn mã lỗi nguyên văn:
function deleteRow(index) {
    let confirm = confirm("Bạn có chắc muốn xóa?");
    if (confirm) {
        listData.splice(index, 0);
        renderTable();
    }
}
2. Đoạn mã đã sửa lỗi hoàn chỉnh:
function deleteRow(index) {
    // CHÚ THÍCH LỖI 1: Trùng tên biến trùng với hàm hệ thống.
    // Không đặt tên biến là 'let confirm' vì nó sẽ đè lên hàm window.confirm() mặc định.
    // Giải pháp: Thay thế tên biến thành 'isConfirmed'.
    let isConfirmed = confirm("Bạn có chắc muốn xóa?");
    
    if (isConfirmed) {
        // CHÚ THÍCH LỖI 2: Sai tham số trong hàm splice().
        // Cú pháp splice(index, 0) nghĩa là xóa 0 phần tử (tức không xóa gì).
        // Giải pháp: Đổi số 0 thành số 1 để xóa chính xác 1 phần tử tại vị trí index.
        listData.splice(index, 1);
        
        // Gọi lại hàm render để cập nhật giao diện bảng ngay lập tức
        renderTable();
    }
}
